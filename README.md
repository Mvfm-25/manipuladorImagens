# manipuladorImagens
Pequeno projeto feito em C++ para manipulção de imagens.

Estava antes simplesmente tentando re-criar um trabalho passado que tive que fazer na cadeira de Programação de Baixo Nível.

Imagino que agora posso já colocar coisas que não foram implementadas anteriormentes.

# Edições possíveis
## 1: Tons de Cinza
Bem auto explicativo, simplesmente torna a imagem preto & branco.
![Alt text](output/B_BW.png?raw=true)

## 2: Cores Invertidas
Torna os valores RGB da imagem o contrário de seu estado original.
![Alt text](output/B_INV.png?raw=true)

## 3: Sépia
Faz com que as imagens ganhem um tom de envelhecidas. Um BW um pouco mais amarelado.
![Alt text](output/B_SEP.png?raw=true)

## 4: Gaussian Blur
Borra as imagens originais, faz com que fosse ver a imagem por meio de um óculos sujo.
![Alt text](output/B_GB.png?raw=true)
